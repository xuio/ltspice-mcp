# Support

## Getting Help

For setup and usage questions, start here:

- [README.md](README.md)
- [AGENT_README.md](AGENT_README.md)
- [COMPATIBILITY.md](COMPATIBILITY.md)

If the issue persists, open a GitHub issue and include:

- MCP server version
- LTspice version
- transport mode
- tool call and parameters
- relevant logs (`tailDaemonLog`, `getRecentErrors`)

## Security Issues

Please avoid posting sensitive details publicly. Share only sanitized logs and artifacts.
